Symfony

Symfony - это MVC-фреймворк с открытым исходным кодом для быстрой разработки современных веб-приложений. Symfony - это полнофункциональный веб-фреймворк. Он содержит набор многократно используемых PHP-компонентов. Вы можете использовать любые компоненты Symfony в приложениях, независимо от фреймворка.Также он обладает огромным функционалом и активным сообществом, имеет гибкую конфигурацию с использованием YAML, XML или аннотаций. Symfony интегрируется с независимой библиотекой и PHP Unit, в основном вдохновлен Ruby on Rails, Django и Spring фреймворками для веб-приложений.

Компоненты Symfony используются во многих проектах с открытым исходным кодом, включая Composer, Drupal и phpBB.Фреймворк Symfony состоит из нескольких компонентов, таких как компонент HttpFoundation, который понимает HTTP и предлагает красивый объект запроса и ответа, используемый другими компонентами. Другие компоненты являются просто вспомогательными, например, Validator, который помогает проверять данные. Компонент Kernel - это сердце системы. Kernel - это, по сути, «главный класс», который управляет окружением и отвечает за обработку http-запросов.Хорошо организованная структура Symfony, чистый код и хорошие практики программирования облегчают веб-разработку. Symfony очень гибкая, на ней можно создавать микросайты и работать с корпоративными приложениями с миллиардами подключений.

Особенности

  • Система на базе Model-View-Controller
  • Высокопроизводительный PHP-фреймворк
  • Гибкая маршрутизация URI
  • Многократно используемый код, который легче поддерживать
  • Управление сессиями
  • Регистрация ошибок
  • Полнофункциональные классы баз данных с поддержкой нескольких платформ
  • Огромное и активное сообщество
  • Набор развязанных и многократно используемых компонентов
  • Стандартизация и совместимость приложений
  • Защита от подделки межсайтовых запросов и других атак
  • Движок шаблонов Twig
  • Микрофреймворк - Symfony можно использовать для разработки конкретной функциональности. Вам не нужно переделывать или устанавливать весь фреймворк
  • Чрезвычайно развитый движок шаблонов и быстрая доставка контента пользователям
  • Совместимость и расширяемость - программисты могут легко расширять все классы фреймворка

    • Интеграции

    Drupal 8 - Drupal это PHP-фреймворк с открытым исходным кодом для управления контентом. Drupal 8 использует основные слои Symfony и расширяет его для поддержки модулей Drupal.

    Thelia - Thelia это решение для электронной коммерции на базе Symfony. Изначально Thelia была написана на PHP-коде и MySQL, однако она не позволяла создавать более быстрые приложения. Чтобы преодолеть этот недостаток, Thelia интегрировалась с Symfony для разработки приложений в настраиваемом виде.

    Dailymotion - Dailymotion один из крупнейших в мире независимых развлекательных видеосайтов, базирующийся во Франции. Приняв решение перейти на фреймворк с открытым исходным кодом и большим сообществом, разработчики Dailymotion решили использовать компоненты Symfony благодаря их гибкости.
